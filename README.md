# zesty-menu

<p align="center"><img src="/img/demo.gif?raw=true"/></p>

Command Line for Zesty Menus

## Install

```bash
> npm install -g zesty-menu
```

## Setup

You must set variable `ZESTY_ID` in your environment to your zesty client_id

## Usage

There are no flags to use

```bash
> zesty-menu
```
